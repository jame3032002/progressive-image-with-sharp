# progressive-image-with-sharp
Example to use sharp module

### Installing
```
npm install
```

### Run to create progressive image
```
node manageImage
```

### Run for example
```
node server
```

#### Go to
http://localhost:5000/

Tip: Enable Slow 3G for test (press f12 > Select Network > Select Slow 3G)
